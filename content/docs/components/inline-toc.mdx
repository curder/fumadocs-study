## 用法 [#usage]

一般将内联目录放在文档的最上方，使用 `InlineTOC` 组件即可。

```jsx title="src/app/docs/[[..slug]]/page.tsx"
import { InlineTOC } from "fumadocs-ui/components/inline-toc"; //[!code ++]

<DocsPage>
  <DocsBody>
    // [!code ++]
    {page.data.toc ? <InlineTOC open items={page.data.toc} /> : null}
  </DocsBody>
</DocsPage>;
```
