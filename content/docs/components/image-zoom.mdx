## 使用 [#usage]

使用 `ImageZoom` 组件替换 `img` 标签即可在全站使用图片缩放。

```tsx title="src/mdx-components.tsx"
import type { MDXComponents } from "mdx/types";

import defaultComponents from "fumadocs-ui/mdx";
import { ImageZoom } from "fumadocs-ui/components/image-zoom"; //[!code ++]

export function getMDXComponents(components?: MDXComponents): MDXComponents {
  return {
    ...defaultComponents,
    img: (props) => <ImageZoom {...(props as any)} />, //[!code ++]
    ...components,
  };
}
```

现在，将在所有图像上自动启用图像缩放。

